# Contributing to RT Controller Framework

Thank you for your interest in contributing to the RT Controller Framework!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Run tests (`colcon test --packages-select rt_controller_framework`)
5. Commit your changes (`git commit -am 'feat: Add my feature'`)
6. Push to the branch (`git push origin feature/my-feature`)
7. Create a Pull Request

## Code Style

- C++17 with ROS2 Humble conventions
- Use `ament_uncrustify` formatting (Allman brace style)
- All files must include the Apache-2.0 copyright header
- Run `ament_cpplint` and `ament_cppcheck` before submitting
